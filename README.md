# **Ticket System**

"Ticket System es una plataforma para registro de tickets para garantia, esta disponible para todos los SO´s, es posible conectarse a Ticket System en
dispositivos como:
Android, iPhone, Windows Phone, PC... etc debido a que usamos el navegador para la conexion.
Debido a que **Ticktet System** usa el mismo plan de datos que utilizas para e-mail o navegacion, no existe costo extra por este sistema de Tickets"

## **Iniciando con Ticket System:**

Para poder empezar a usar Ticket System solo sigue estas simples instrucciones:

1. Accede a **URL del sitio**.  
2. Dirigete al hasta abajo en la pagina principal y encontraras el formulario de registro.  
3. Ingresa tus datos tales como:   
    - Nombre  
    - Correo  
    - Telefono  
    - Numero de Ticket de Compra  
    - Imagen  
    - Comentario  
4. Click en el boton de aceptar, para que tus datos sean registrados correctamente.  
5. Una vez registrados tus datos, te redirigira a la agina de exito!!! y te enviaremos un mensaje a tu correo.  
6. Una vez revisado tu equipo te enviaremos el diagnostico, y estarmeos en contacto contigo para darte mas detalles hasta el fina del proceso.  

### **Requisitos**

Para poder utilizar Ticket System solo necesitas un navegador web compatible con : HTML5.
